# 💸 Expense Prediction Pro

A smart and modern **Expense Management Web Application** built using **Django** that helps users track daily expenses, manage budgets, visualize spending patterns, and download PDF reports.

🚀 Live Project: *(Add your Render link here)*

## ✨ Features

* User Registration & Login Authentication
* Add, Edit, Delete Expenses
* Premium Dashboard with Summary Cards
* Expense Visualization using Pie Charts
* Download Expense Reports in PDF
* Budget Tracking System
* Expense Prediction Overview
* Responsive UI Design
* Deployed Online using Render

## 🛠️ Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* JavaScript
* Chart.js
* ReportLab
* WhiteNoise
* Gunicorn
* Render
